# Ai Mobile & Computers — Website

A modern, mobile-first product catalog website for **Ai Mobile & Computers** electronics shop.

## 🌟 Features

- **Dynamic Product Catalog** — Products load from Google Sheets (or fallback data)
- **Category Filtering** — Filter by Mobile, Computer, or CCTV accessories
- **Product Search** — Real-time search across all products
- **WhatsApp Integration** — One-click inquiry with pre-filled product messages
- **Mobile-First Design** — Optimized for smartphones with responsive layout
- **Lazy Loading** — Images load on scroll for fast page speed
- **Dark Theme** — Premium electronics store aesthetic

## 📁 File Structure

```
├── index.html          # Main page
├── css/styles.css      # Custom styles
├── js/
│   ├── app.js          # UI logic, navigation, animations
│   ├── products.js     # Product fetching, filtering, rendering
│   └── data.js         # Fallback static product data
├── images/             # Images and assets
└── GOOGLE_SHEET_SETUP.md  # Guide for the shop owner
```

## 🚀 Deployment to GitHub Pages

1. Push all files to your GitHub repository
2. Go to **Settings → Pages**
3. Under **Source**, select **Deploy from a branch**
4. Choose **master** branch, **/ (root)** folder
5. Click **Save** — your site will be live in ~2 minutes!
